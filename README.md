# 📂 Portafolio de Automatización de Pruebas Mobile - Jhoselin Luizaga 

Este repositorio alberga los proyectos de automatización de pruebas Mobile que desarrollé durante la mentoría QA Pro Level de QAX. Mi objetivo es demostrar mis habilidades para crear, ejecutar y mantener tests robustos y confiables, garantizando la mejor calidad.

# 🤖 Tecnologías utilizadas
- Java
- Maven
- Appium
- Android Studio

# Proyectos

## 🎯 Stage 1:

Instalación del JDK, configuración del Android SDK, creación de emuladores Android y preparación del proyecto con Maven, integrando las dependencias necesarias para Appium, y Cucumber
- Instalación y configuración del entorno Android
- Creación y configuración de emuladores
- Introducción a Appium
- Primera automatización Android con Appium

### Proyectos: 

- Quick Task
- Challenge
- Mission

## 🎯 Stage 2:

Localizar elementos básicos dentro de una aplicación móvil, entendiendo cómo inspeccionarlos correctamente con Appium Inspector y aplicando buenas prácticas para seleccionar localizadores efectivos

- Localizadores basicas en Android
- Implementación Page Object Model (POM)
- Acciones basicas en Android
- Validaciones y aserciones

### Proyectos: 
- Quick Task
- Challenge
- Mission
